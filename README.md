# 🚀 AI & Machine Learning Projects Portfolio

## 👩‍💻 Overview

This repository contains a collection of my hands-on projects in **Artificial Intelligence, Machine Learning, and Generative AI**.
The focus is on building **end-to-end systems** — from data preprocessing to model development and deployment.

Each project demonstrates practical problem-solving using real-world datasets and modern AI techniques.

---

## 📌 Projects Included

### 1. 🧠 AI-Powered Sentiment Analysis & NLP Pipeline

* Built an end-to-end NLP pipeline for sentiment classification
* Applied preprocessing techniques: tokenization, stopword removal, lemmatization
* Used TF-IDF vectorization with Logistic Regression & Naive Bayes
* Achieved ~88% accuracy on review datasets
* Deployed using Streamlit for real-time predictions

---

### 2. 🔍 RAG-Based Knowledge Assistant

* Implemented a Retrieval-Augmented Generation (RAG) system
* Combined document retrieval with response generation
* Designed modular pipeline for query → retrieval → answer generation
* Focused on improving contextual relevance and reducing hallucination

---

### 3. 📊 Customer Churn Prediction

* Built a classification model to predict customer churn
* Performed EDA, feature engineering, and model training
* Used Random Forest and evaluated using ROC-AUC (~0.91)
* Visualized feature importance and model performance
