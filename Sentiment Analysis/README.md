# AI-Powered Sentiment Analysis & NLP Pipeline

## 🚀 Overview

This project implements an end-to-end NLP pipeline to classify customer sentiments (positive/negative/neutral) using machine learning techniques.

## 🧠 Features

* Text preprocessing (tokenization, stopword removal, lemmatization)
* TF-IDF vectorization
* Logistic Regression model
* Streamlit web interface

## 🛠️ Tech Stack

Python, NLTK, Scikit-learn, Streamlit

## ▶️ Run Project

```bash
pip install -r requirements.txt
python train.py
streamlit run app.py
```

## 📊 Results

Achieved ~88% accuracy on review datasets.

## 📌 Learnings

* NLP preprocessing pipeline
* Feature extraction using TF-IDF
* Model deployment using Streamlit
